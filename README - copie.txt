----------
To use :

Start Mamp (host MySQL & Wordpress)
Start Server 


----------
Initiate Vue project :

Create project : vue create Mycelium
Go to project folder : cd [Mycelium]
Install Vue Router : npm install vue-router@4
Create Routes : npm install create-routes --save
Install Vue X : npm install vuex@next --save
Install SCSS : npm install --save-dev node-sass sass-loader
Install Cookies: npm install vue-cookies --save

Run server : npm run serve


----------
Start mySQL :

- Start MAMP
- Open Terminal
- /Applications/MAMP/Library/bin/mysql -u root -p  (password "root")

- - - - -
Use SQL database :

- source [chemin]
- use [file]


----------
Wordpress :

Run on MAMP server
File installed in /Applications/MAMP/htdocs/wordpress

(Installed on Wordpress for Authentication) WordPress REST API Authentication


----------
Manage database (online HMI)

- http://localhost/phpMyAdmin5/


----------
Start server :

- npm run serve
